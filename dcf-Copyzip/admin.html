Scripts de Publicidade (Adsterra / Ezoic)

Cole aqui o código completo (incluindo <script>...</script>) fornecido pela Adsterra ou Ezoic. Os scripts são injectados dinamicamente nos slots correctos da app.

Aparece na home, dentro do cartão de bónus de boas-vindas (após reclamado).

Aparece em cada anúncio do ecrã "Ver Anúncios" (recompensa de 0.05 MT por anúncio).

Sistema de Convites

O domínio é usado para construir o link de convite no formato dominio/CODIGO. Cada utilizador recebe um código único de 7 caracteres no momento do registo. Aqui podes também gerar códigos extra (para campanhas, promoções, etc.) que ficam disponíveis para qualquer pessoa usar até serem reclamados.

Os links de convite ficam: https://mozpay.co.mz/?code=ABC1234
